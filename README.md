## 🛠️ How to Compile & Run

## 💡 Pro-Tip
> Make sure you have `gcc` (GNU Compiler Collection) installed on your system.

### Compiling a Single File
```bash
gcc -o output_name filename.c
```

### Running the Executable
```bash
./output_name
```
